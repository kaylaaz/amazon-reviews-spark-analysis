# Amazon US Customer Reviews Spark Analysis

## Project Overview
This project analyzes the Amazon US Customer Reviews dataset from Kaggle, which contains over 50 GB of reviews written between 1995 and 2015 across 37 product categories. Each record includes fields such as helpful votes, product category, review date, review text, star rating, total votes, and verified purchase status. 

We will use Spark for distributed processing to investigate what makes a review useful to other shoppers. We examine what factors relate to a review's helpfulness ratio (helpful votes divided by total votes) across different product categories. 

Our goal is to identify patterns that separate valuable reviews from those that are ignored, which will provide insight into which information shoppers find most useful.

## Dataset
Amazon US Customer Reviews Dataset (Kaggle): https://www.kaggle.com/datasets/cynthiarempel/amazon-us-customer-reviews-dataset
